# Sudoku Clone

## Difficulties
- **EASY - 30 EMPTY SPACES**
- **MEDIUM - 40 EMPTY SPACES**
- **HARD - 55 EMPTY SPACES**

## How to play
- **MOVE**: ARROW keys or mouse hover/click
- **SKETCH**: Press number once
- **LOCK IN NUMBER**: Press number again or press ENTER
